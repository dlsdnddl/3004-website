## 📋 1단계: 네이버 서치어드바이저 등록

### 1-1. 사이트 등록
1. **네이버 서치어드바이저 접속**: https://searchadvisor.naver.com/
2. **로그인**: 네이버 계정으로 로그인
3. **웹마스터 도구**: 좌측 메뉴에서 선택
4. **사이트 등록**: `https://3004.co.kr` 입력

### 1-2. 사이트 소유 확인
**방법 1: HTML 파일 업로드 (추천)**
1. 네이버에서 제공하는 HTML 파일 다운로드 (예: `naver1234567890.html`)
2. `/home/user/webapp/public/` 폴더에 업로드
3. 배포 후 `https://3004.co.kr/naver1234567890.html` 접근 가능 확인
4. 네이버에서 "소유 확인" 클릭

**방법 2: 메타태그 추가**
1. 네이버에서 제공하는 메타태그 복사
2. `/home/user/webapp/src/renderer.tsx` 파일의 `<head>` 섹션에 추가:
```tsx
<meta name="naver-site-verification" content="네이버에서_제공한_코드" />
```
3. 배포 후 "소유 확인" 클릭

### 1-3. 사이트맵 제출
1. 네이버 서치어드바이저 → **요청** → **사이트맵 제출**
2. 사이트맵 URL 입력: `https://3004.co.kr/sitemap.xml`
3. **확인** 클릭

### 1-4. RSS 제출 (선택)
- 블로그가 있다면 RSS 피드도 제출

### 1-5. 수집 요청
1. **요청** → **페이지 수집 요청**
2. 주요 페이지 URL 입력 후 제출:
   - `https://3004.co.kr/`
   - `https://3004.co.kr/visit-care`
   - `https://3004.co.kr/family-care`
   - `https://3004.co.kr/bath-service`
   - `https://3004.co.kr/welfare-equipment`
   - `https://3004.co.kr/grade-application`
